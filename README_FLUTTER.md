# StudySync

忘却曲線にもとづいて復習タスクを自動で組み立てる、個人開発の学習管理アプリです。
Flutter製、Android / Web対応（iOSは今後対応予定）。

## どんなアプリ？

タスクを完了にすると、設定した間隔（既定：1日後・3日後・7日後・14日後・30日後）で
復習タスクが自動的に追加されます。「いつ何を復習すればいいか」を考える手間を
なくすことが、このアプリの一番の目的です。

### 主な機能

- **タスク管理** — 未完了・完了・期限切れ・復習のみ・締切が近い順、で一覧を切り替え
- **忘却曲線にもとづく自動復習** — タスク完了時に、設定した間隔で復習タスクを自動生成
- **教科・プロジェクト** — 「教科」（数学・英語など）でタスクを分類し、さらに
  「プロジェクト」（期末試験・小テスト対策などの任意タグ）でも横断的に絞り込み可能
- **学習タイマー** — 計測した時間はタスクごとの学習記録として保存・あとから削除も可能
- **カレンダー** — 月表示でタスクの締切を一覧、教科の色で表示
- **統計** — 日別・週別・教科別・タスク別の学習時間グラフ
- **通知** — タスクごとに「期限の何分前に知らせるか」を設定可能（復習・通常タスクで別設定）
- **Googleアカウント連携** — Firestoreでのクラウド同期、複数端末で同じデータを共有
- **カスタマイズ** — 壁紙、文字サイズなど

### 技術スタック

Flutter / Dart・Provider（状態管理）・Firebase（Authentication, Cloud Firestore）・
flutter_local_notifications・SharedPreferences（ローカル保存）

## バージョン履歴

`lib/changelog.dart` の内容と同じです（アプリ内では 設定 → アプリ情報 から見られます）。

### v0.5
- Googleアカウントを切り替えても前のアカウントのタスクが残って見える不具合を修正（連携を解除すると、表示が初期状態に戻るように変更）
- 教科のカラーを、RGBスライダー＋HEXコードで選べるカスタムカラーピッカーに変更
- 壁紙のカラーにも、同じカスタムカラーピッカーで好きな色を追加できるように（追加した色は長押しで削除可能）

### v0.4
- 通知が予約だけ届かない不具合を修正（AndroidManifest.xmlに、通知の予約を受け取るためのreceiverが登録されていなかったのが原因でした）
- 設定 →「プロジェクト」から、タスク作成時に出てくる候補（旧グループ）を追加・名前変更・削除できるように
- タスク作成時の「プロジェクト」欄を、教科と同じドロップダウン選択方式に変更
- ホームのドロワーから「プロジェクト」でも絞り込めるように
- カレンダー：選択中の日（青色の背景）と通常タスクのドット（同じ青系の色）が重なって見えにくい問題を修正
- カレンダー：1日に表示されるタスクの点が最大3つに制限されていたのを廃止し、実際のタスク数と一致するように修正
- カレンダーのタスクの点の色を、状態（期限切れ等）ではなく教科の色に合わせるように変更
- 設定の「UIの配置」（ナビゲーションの並び替え）を削除

### v0.3
- 計測中のタイマー画面：×ボタンが反応しない不具合を修正し、下にスワイプしても閉じられるように変更
- 通知が届かない不具合を修正（端末の設定から直接許可した場合も、許可状態を正しく認識するように）
- 元のタスクを削除すると、そこから自動生成された復習タスクも一緒に削除
- 教科の「テスト期間（開始・終了日）」機能を削除
- 用語を変更：「プロジェクト」→「教科」、「グループ」→「プロジェクト」
- 初回起動時のデモタスクを3件程度に削減
- 設定 → アプリ情報から、バージョンごとの変更履歴が見られるように

### v0.2
- アプリアイコンを変更、アプリ名を「StudySync」に変更
- 起動時に、忘却曲線が描かれるアニメーションを追加
- 学習記録（タイマーのセッション）を個別に削除できるように
- タスク一覧に「締切が近い順」を追加
- 通知を再設計：タスクごとに「期限の何分前に知らせるか」を設定する方式に変更
- グループ名を一度使うと、次回から候補として表示され続けるように
- 新規タスクの期限の既定値を「今日の23:00」に変更（設定で変更可能）

### v0.1
- 初回リリース：タスク管理、忘却曲線にもとづく自動復習、学習タイマー、統計、
  カレンダー、Googleアカウント連携、通知、配布ページ
