StudySync

忘却曲線にもとづいて復習タスクを自動管理する学習アプリです。
下のボタンからダウンロードしてください。

🤖

Android版

apkファイルを直接ダウンロードしてインストールします。初回のみ「不明なソースからのインストール」を許可する必要があります。

⬇ Androidでダウンロード
  1. 上のボタンをタップしてapkをダウンロード
  2. ダウンロード通知をタップして「インストール」
  3. 初回は「このソースを許可」の確認が出たらON
🍎

iOS版（iPhone / iPad）

TestFlightアプリ経由で配布します。下のリンクから参加すると、TestFlightアプリのインストール → StudySyncのインストールへ進めます。

🔗 TestFlightリンク（準備中）
  1. App Store Connectでテスターとして招待される
  2. TestFlightアプリ（App Storeにあります）をインストール
  3. 招待リンクを開き、StudySyncをインストール

このページは GitHub Pages で公開されています。Android版は誰でもダウンロードできますが、 iOS版（TestFlight）は招待されたテスターのみインストールできます。
